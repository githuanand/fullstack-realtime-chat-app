# Full Stack Real-Time Chat Application

Built a production-ready Full Stack Real-Time Chat Application using React.js, FastAPI, PostgreSQL, JWT Authentication, and WebSockets.

## Tech Stack
### Frontend:
- React.js (Vite)
- Tailwind CSS
- React Router DOM
- Axios
- Zustand
- Framer Motion

### Backend:
- FastAPI
- PostgreSQL (Neon)
- SQLAlchemy
- JWT Authentication
- bcrypt
- WebSockets

## Features
- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Real-Time Messaging
- User Search
- Online/Offline Presence
- Responsive UI

## Project Structure
client/ → Frontend  
server/ → Backend
